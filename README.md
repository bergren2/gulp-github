## Prereqs

- [Yarn](https://yarnpkg.com/en/docs/install)
  - `$ brew install yarn`
  - `$ choco install yarn -y`

## Setup

    $ yarn install
    $ yarn run setup

After you've tweaked `config.json` with the [correct GitHub credentials](https://github.com/settings/tokens), do

    $ gulp
